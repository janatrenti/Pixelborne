\hypertarget{class_enemy_actions}{}\doxysection{Enemy\+Actions Class Reference}
\label{class_enemy_actions}\index{EnemyActions@{EnemyActions}}


Is attached to enemy game objects in order to simply let them execute actions that are defined in \mbox{\hyperlink{interface_i_enemy_actions}{I\+Enemy\+Actions}}. It takes care of the animations, physics and health. It is intended to be used with an \mbox{\hyperlink{class_action_pattern_executor}{Action\+Pattern\+Executor}} attached to the same game object. Though it can be used without it.
