Is implemented by \mbox{\hyperlink{class_singleplayer}{Singleplayer}} and \mbox{\hyperlink{class_multiplayer}{Multiplayer}} class and defines the common methods. It is important to note that the game-\/instances \mbox{\hyperlink{class_singleplayer}{Singleplayer}} and \mbox{\hyperlink{class_multiplayer}{Multiplayer}} work closely together with the camera. When a player dies it tells the active game that it dies. This initiates a fade out in the camera. When the camera finished the fade out it notifies the game and the game can take further actions e.\+g. changing the multiplayer stage and fading in again. Fading in has the same communication structure between the camera and the game.
